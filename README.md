# Movie Search Engine

## Описание
MSE - небольшой проект, который позволяет искать фильмы по названию.
Благодаря проекту можно получить информацию о необходимом фильме, такую как:
1. Название
2. Постер
3. Режиссер
4. Актерский состав
5. Описание
